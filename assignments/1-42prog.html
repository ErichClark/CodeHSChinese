1.42 Take Em All 採取全身所有

Objective 目標:

There is a stack of 100 tennis balls in front of karel. Have karel move to the stack, take all of the tennis balls, and then move one more time. Use a for loop!
有一疊100網球在卡雷爾的前面。有卡雷爾移動到堆棧中，把所有的網球，然後再移動一次。使用為循環！

Starting World and Ending World 世界開始和結束的世界

For Loop Hints 為循環提示

  /* Repeat code betweeen   之間的重複代碼
  * brackets COUNT times */   括號算時
}
