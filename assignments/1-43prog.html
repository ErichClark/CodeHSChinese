Program 1.43 Dizzy Karel 暈卡雷爾

Objective 目標:

Write a program to have karel spin in 8 circles. This means karel is going to need to turn left 32 times. You must use a for loop in this program.
編寫一個程序，有卡雷爾旋在8圈。這意味著卡雷爾是將需要左轉32次。你必須使用的循環此計劃。
