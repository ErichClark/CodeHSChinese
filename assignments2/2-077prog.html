2.77 Assignment: Inventory 作业：库存

Objective 目標:

Write a program that keeps track of a simple inventory for a store. While you still have items left, ask the user how many items they would like to buy. Then print out how many they have left. You should use a while loop for this problem. A sample run is below.
編寫一個程序，跟踪一個簡單的庫存的商店。雖然你仍然有物品遺留，問他們有多少項目，如購買用戶。然後打印出他們有多少離開了。 你應該使用一個while循環這個問題。一個運行示例如下。
