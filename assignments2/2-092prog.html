2.92 Assignment: Height in Meters 作业：身高米

Objective 目標:

Write a function that takes a height in feet and inches and converts it to meters.
寫一個函數，它接受英尺和英寸的高度，並將其轉換為米。

Provided Constants 提供的常量 

var INCHES_TO_CM = 2.54;
var CM_TO_METERS = 0.01;

變量 英寸_以_厘米 = 2.54;
變量 厘米_以_米 = 0.01;

Sample Run 樣品運行： 

For example, if you are 6 feet, 4 inches tall, then calling

	convertHeightToMeters(6, 4);
should print 1.9304 meters. With all these calls, you should get:

	convertHeightToMeters(6, 4);
	convertHeightToMeters(5, 8);
	convertHeightToMeters(5, 2);
6 feet, 4 inches = 1.9304 meters
5 feet, 8 inches = 1.7272 meters
5 feet, 2 inches = 1.5748 meters

舉例來說，如果你有6個英尺4英寸高，然後調用 

	轉換高度為米(6, 4);
應打印1.9304米。所有這些要求，你應該得到的： 

	轉換高度為米(6, 4);
	轉換高度為米(5, 8);
	轉換高度為米(5, 2);
6 腳, 4 英寸 = 1.9304 米
5 腳, 8 英寸 = 1.7272 米
5 腳, 2 英寸 = 1.5748 米
