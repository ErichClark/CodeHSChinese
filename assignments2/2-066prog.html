2.66 Assignment: Factorial 分配：阶乘

Objective 目標:

Write a program that prints out a factorial. N factorial (written N!) is defined as   編寫打印出一個階乘的程序。 N個因子（寫為n！）被定義為
N * (N-1) * (N-2) * ... * 1
For example, 4! is   例如，4！是
4 * 3 * 2 * 1 = 24.
For a given constant N, use for loops and variables to print out N factorial. Hint: You solve factorial by multiplying the numbers from 1 all the way up to N together. You need a variable to store the result.
對於給定的常數N，使用for循環和變量打印出N個因子。 提示：您可以通過從1一路攀升至N乘以數相加解決階乘。你需要一個變量來存儲結果。
