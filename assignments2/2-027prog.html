2.57 Assignment: Caterpillar 分配：卡特彼勒

Objective 目標:

This program should draw a snowman. The snowman should have three sections, a bottom snowball with radius BOTTOM_RADIUS, a middle snowball with radius MID_RADIUS, and a top snowball with radius TOP_RADIUS. The circles should all be gray, and you can set the color using Color.gray. The snowman should be centered horizontally in the world, and should be sitting on the bottom of the canvas (Hint: You will need to use getHeight() here).
If you are feeling adventurous, can you give the snowman a hat, or eyes?
