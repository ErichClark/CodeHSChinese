2.106 Assignment: Max 赋值：最大

Objective 目標:

Write a function called `max` that returns the maximum of the two numbers passed in as parameters.
寫了一個名為'最大`函數，返回作為參數傳入的兩個數的最大。

Sample Run 樣品運行： 

For example, if you made a call like
   var x = max(10, 14);
x should have the value 14.

舉例來說，如果你喜歡做一個電話 
	變量x=最大(10, 14);
x應該具有的價值14。
