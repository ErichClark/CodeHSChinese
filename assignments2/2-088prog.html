2.88 Assignment: Triple 分配：三重

Objective 目標:

Write a function called `triple` that triples a number and prints it out. Then in your start function, make several calls to the triple function to test it out.
寫了一個名為'三`功能三倍一些並打印出來。然後在你的啟動功能，使多次打電話給三重功能來測試它。

Sample Run 樣品運行： 

For example, if you made a call like
triple(4)
it should print out 12.

舉例來說，如果你喜歡做一個電話 
三（4） 
它應該打印出12。
