2.38 Assignment: The Weekend 分配：本周末

Objective 目標:

Write a program that asks the user first if it is a Saturday, and stores that in a boolean variable. Then ask the user if it is a weekday and store that in a different variable. Then try to figure out if it is the weekend. (Hint: It is the weekend if it is a Saturday or if it is not a weekday).
問兩個布爾值的用戶。首先應該被存儲在一個變量算命不管他們是否有足夠的單位（學分），第二個應該被存儲在一個變量算命無論他們是否已符合要求。 他們只能畢業，如果他們有足夠的單位，並已符合要求。打印出他們是否能畢業。
