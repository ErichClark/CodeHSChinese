2.102 Assignment: Triple with Return Values 分配：三重与返回值

Objective 目標:

Write a function called `triple` that triples a number and returns the result. Then in your start function, make several calls to the triple function to test it out.
寫了一個名為'三`功能三倍數字並返回結果。然後在你的啟動功能，使多次打電話給三重功能來測試它。

Sample Run 樣品運行： 

For example, if you made a call like
	var x = triple(4);
then x should hold the value 12.

舉例來說，如果你喜歡做一個電話 
	變量x=三（4）; 
那麼x應持有​​的價值12。
